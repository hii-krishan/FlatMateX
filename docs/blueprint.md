# **App Name**: FlatMateX

## Core Features:

- Expense Manager: Add, edit, and divide shared expenses and track rent with charts. Expenses are stored in MongoDB.
- AI Room Assistant: A chat or voice-based assistant using OpenAI API that provides reminders, generates to-do lists, and suggests budget insights using the tool and the information it receives.
- Nearby Services Finder: Locate nearby services (tiffin centers, laundry, etc.) using Google Maps API. This utilizes MongoDB.
- Task Planner: Add class timetable, daily study planner, and task checklist with reminders. Uses MongoDB to store this data.
- Smart Grocery Tracker: Shared grocery list with restock reminders and AI-generated suggestions, all stored in MongoDB.
- Mood Tracker: Track daily mood, sleep, and productivity, with self-care advice from AI using a tool to analyze moods.. The application is connected to the MongoDB database.
- Event Planner: Create and share events, polls, and reminders stored in MongoDB.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a calm and organized feel.
- Background color: Light indigo (#E6E6FA) with a desaturated look, for a calm and easy to use feel.
- Accent color: Violet (#8F00FF) to draw attention to key interactive elements.
- Headline font: 'Space Grotesk' sans-serif for a computerized and techy look. Body font: 'Inter' sans-serif for a modern neutral look.
- Code Font: 'Source Code Pro' for computer code snippets
- Use consistent, minimalist icons for navigation and key features.
- Subtle transitions and animations to enhance user experience when the data shown in the app changes.
